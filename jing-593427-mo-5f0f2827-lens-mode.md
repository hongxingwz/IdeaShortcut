# '镜头'模式

将鼠标的指针悬浮在一个警告，错误条纹或一些在滚动盒之外的滚动条上。IDEA展示warning/error信息注解的源代码上。

这帮助你浏览用于记号的上下文，和超过编辑器可视区域的原代码。

![](/assets/1506088602688.png)

此模式默认被启用

**切换镜头模式**
*做下面的其中一个：
    * 打开Appearance的设置框
    * Right-click the code analysis marker on top of the current editor

* 选择或清除Show code lens on scrollbar hover.
