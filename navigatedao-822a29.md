Alt + Up/Down

### 方法继承结构

Control + Shift + H

### 类继承结构

Control + H

### 方法调用结构

Control + Alt + H

### 展示该类的方法

Control + F12

## 导航

| action | description |
| --- | --- |
| 最近打开的文件\(Recent Files\) | Ctrl + E |
| 导航类\(Navigate to class\) | Ctrl + N |
| 导航文件\(Navigate to File\) | Ctrl + Shift + N |
| 导航变量\(Navigate to Symbol | Ctrl + Shift + Alt + N |
